Opalus UI uses the shadcn registry format and supports projects configured with
either Base UI or Radix.

## 1. Initialize shadcn

Choose your preferred component base when creating `components.json`:

```bash
pnpm dlx shadcn@latest init --base base
```

Use `--base radix` instead if your project uses Radix. Opalus UI resolves the
matching shadcn dependencies automatically.

## 2. Add the registry

Add the production registry to `components.json`:

```json
{
  "registries": {
    "@opalus-ui": "https://opalus-ui.miyuzu.top/r/{name}.json"
  }
}
```
