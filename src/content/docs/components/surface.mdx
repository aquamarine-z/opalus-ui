Surface provides promise-based dialogs over the shadcn Dialog installed in the
consumer project. The same registry item supports Base UI and Radix without a
different import or install command.

## Installation

<InstallationGuide componentName="surface" client:load/>

Render exactly one `NiceModal.Provider` above every component that opens a
surface:

```tsx
import NiceModal from "@ebay/nice-modal-react"

export function App({ children }: { children: React.ReactNode }) {
  return <NiceModal.Provider>{children}</NiceModal.Provider>
}
```

## Shared Options

```tsx
type CustomDialogOptions = {
  modal?: boolean
  closeOnOverlayClick?: boolean
}

type DialogOptions = CustomDialogOptions & {
  showCloseButton?: boolean
  title?: React.ReactNode
  titleIcon?: React.ReactNode
  showTitleIcon?: boolean
}
```

`modal` and `closeOnOverlayClick` default to `true`. A non-modal dialog only
closes through the `close` callback or an Opalus action. When overlay closing is
disabled, Escape remains available for modal dialogs.

## Custom Dialog

```tsx
dialog.custom<T>(
  content: (close: (result?: T) => Promise<void>) => React.ReactNode,
  options?: CustomDialogOptions
): Promise<T | undefined>
```

Return `SurfaceDialogContent` as the root of custom content when using Opalus
close controls or `closeOnOverlayClick: false`.

## Alert Dialog

```tsx
type AlertDialogOptions = DialogOptions & {
  message?: React.ReactNode
  closeButtonContent?: React.ReactNode
}

dialog.alert(options?: AlertDialogOptions): Promise<null | undefined>
```

## Confirm Dialog

```tsx
type ConfirmDialogOptions = DialogOptions & {
  message?: React.ReactNode
  confirmButtonContent?: React.ReactNode
  cancelButtonContent?: React.ReactNode
}

dialog.confirm(options?: ConfirmDialogOptions): Promise<boolean | undefined>
```

The promise resolves to `true` for Confirm, `false` for Cancel, and `undefined`
for an automatic close.

## Prompt Dialog

```tsx
type PromptDialogOptions = DialogOptions & {
  message?: React.ReactNode
  defaultValue?: string
  placeholder?: string
  confirmButtonContent?: React.ReactNode
  cancelButtonContent?: React.ReactNode
}

dialog.prompt(options?: PromptDialogOptions): Promise<string | null | undefined>
```

The promise resolves to the entered string, `null` for Cancel, or `undefined`
for an automatic close.
