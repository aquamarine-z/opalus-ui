`TimePicker` uses the shadcn Slider installed for the consumer's selected Base UI
or Radix configuration. Wrap multiple pickers in `TimePickerContainer` to receive
their combined duration in milliseconds.

## API

| Prop | Type | Default |
| --- | --- | --- |
| `timeMilliseconds` | `number` | `1` |
| `minValue` | `number` | `0` |
| `maxValue` | `number` | `1000` |
| `step` | `number` | `1` |
| `value` | `number` | uncontrolled |
| `defaultValue` | `number` | `minValue` |
| `onValueChange` | `(value: number) => void` | `undefined` |
| `prefixLabel` | `ReactNode \| (value) => ReactNode` | `undefined` |
| `suffixLabel` | `ReactNode \| (value) => ReactNode` | `undefined` |
